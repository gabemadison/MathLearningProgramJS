Welcome to the math learning program, written in JavaScript. This code will hopefully be useful for myself as well as others in the future (to be possibly put onto a website?)

Goals for this project include:

  • Text box for a person to enter their responses to the questions.
  • Ability to answer addition, subtraction, multiplication, and division questions.
  • Ability to select the number of questions to practice on.
  • Ability to answer binary conversion questions.
  • Ability to answer hexadecimal conversion questions.
  • Ability to answer bit manipulation questions.
  • Ability to show correct answers for the questions (immediately after answering).
  • Ability to show the steps to correctly solve the problem.

For a comprehensive list of all the changes and improvements we've made to the Math Learning Program, please visit our changelog page.
